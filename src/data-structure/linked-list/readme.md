# Linked-List

## 数据结构

每个元素由一个存储元素本身的节点和一个指向下一个元素的引用（也称指针或链接）组成。
![](../../../assets/images/linked-list/1.png)

### 链表尾部添加元素

```ts
while (curr.next != null) {}
// curr为最后一项
```
